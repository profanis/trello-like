# A Trello Like app with Angular and GraphQL

![A Trello Like app with Angular and GraphQL](client/src/assets/ezgif.com-gif-maker.gif)

## Stack

### Client

- Angular
- GraphQL w/ Apollog Client
- Angular CDK
- Bootstrap CSS
- Icons: https://feathericons.com/

### Server

- NodeJs
- Express
- Apollo Server
- MongoDB
- Mongoose
